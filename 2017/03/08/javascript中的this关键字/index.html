javascript(缩写js)语言中的this和java,php中的this是完全不同的概念。
js中的this是动态的，总是指向一个对象，但是这个对象是谁，需要根据函数的调用上下文来决定。

  2. 作为普通的函数来调用(默认绑定)
  3. 作为构造器来调用
  4. call和apply方法调用
